# 基于SadTalker实现的数字人视频生成应用
1.	git clone https://github.com/Kieeru/digital_human.git
2.	在[SadTalker](https://github.com/OpenTalker/SadTalker)的[README](https://github.com/OpenTalker/SadTalker/blob/main/README.md) 中下载预训练模型和GFPGAN补丁，并将其复制到根目录下的checkpoints与gfpgan文件夹中
3.	后续部署查看[README_app](https://github.com/Kieeru/digital_human/blob/main/README_app.md)或[README_app_sadtalker](https://github.com/Kieeru/digital_human/blob/main/README_app_sadtalker.md)
